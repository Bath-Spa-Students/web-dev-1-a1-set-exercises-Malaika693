Html & CSS codes

Html properties

The <html> Represents the entire html document & contains other html elements.

The <Head> Contains meta information about the documnet.

The <Body> Contains the content. including text, images, links.

The <h1> Represents the main heading of the section & there are subheadings as h2, h3, h4.

The<P> Is for paragraghs of text

The <a> Is an anchor tag used to create hyperlinks

The <ul> and <li> Define unordered & odered list

CSS properties

The color Is to set the color for the content.

The margin Is for controlling the space outside the border.

The background-color Is for background color of the element.

The font size Is to specify the size of the font.

The border Is to set border properties of an element such as color, width, & style
